# Ramawatar Portfolio

🚀 Modern Laravel Portfolio Website

## 🔥 Features
- Dynamic Resume Website
- Admin Panel (Add/Edit Projects, Skills, Education)
- Laravel + Tailwind CSS
- Clean UI Design

## 🛠️ Tech Stack
- PHP (Laravel)
- MySQL
- Tailwind CSS
- Blade Templates

## 📂 Sections
- About Me
- Skills
- Education
- Projects
- Contact

## ⚡ Setup
```bash
git clone https://github.com/RamawatarBhakar/ramawatar-portfolio.git
cd ramawatar-portfolio
composer install
cp .env.example .env
php artisan migrate
php artisan serve
